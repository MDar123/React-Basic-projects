        <h2>Getting Started</h2>
        <p>To explore or modify these projects locally, follow these steps:</p>
        <ol>
            <li>
                <p><strong>Clone the Repository:</strong></p>
                <pre><code>git clone https://github.com/MDar123/React-Basic-projects.git</code></pre>
            </li>
            <li>
                <p><strong>Navigate to a Project Directory:</strong></p>
                <pre><code>cd React-Basic-projects/Project_Name</code></pre>
                <p>Replace <code>Project_Name</code> with the desired project's folder name (e.g., <code>Currency_converter</code>).</p>
            </li>
            <li>
                <p><strong>Install Dependencies:</strong></p>
                <pre><code>npm install</code></pre>
            </li>
            <li>
                <p><strong>Run the Application:</strong></p>
                <pre><code>npm start</code></pre>
                <p>The application will launch in your default web browser at <code>http://localhost:3000</code>.</p>
            </li>
        </ol>
